Enable restricted access for logged in users

== Description ==

Activate a feature on your site where only logged in users will be able to browse pages and posts and non logged in users will be redirected to the restrict-access page.

== Installation ==

1. Install the Fix Restrict Access plugin either via the WordPress plugin directory, or by uploading the files to your web server (in the `/wp-content/plugins/` directory).

2. Activate the Fix Restrict Access plugin through the 'Plugins' menu in WordPress.
